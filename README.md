# 🌲 My Cypress Automation

This project utilizes **Cypress** to automate the testing of a **Login** feature and a **User Update** feature. Cypress is a JavaScript-based end-to-end (E2E) testing framework that allows developers to write tests and execute them in a real browser.

---

## 🛠️ Prerequisites

Before you begin, ensure you have the following installed:
* **Node.js** (LTS version recommended)
* A terminal of your choice (Bash, CMD, PowerShell)

> [!IMPORTANT]  
> It is necessary to have **Node.js** installed globally on your machine to manage dependencies.

---
